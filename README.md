# Playwright MCP

## Description

This project uses Playwright MCP (Mission Control Protocol) for browser automation and testing.

## Installation

First, install the required dependency:

```bash
npm install @playwright/mcp
```

## Usage

To start the Playwright MCP server, run:

```bash
npx @playwright/mcp@latest --port 8931
```

This will start the server on port 8931, which you can then connect to for browser automation tasks.

## Requirements

- Node.js (v14 or higher recommended)
- npm

## Documentation

For more information about Playwright MCP, visit the [official documentation](https://playwright.dev/).
